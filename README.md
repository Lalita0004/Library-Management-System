# Library Management System

## Overview

The Library Management System is a software application developed to manage library operations efficiently. It helps in maintaining book records, issuing and returning books, and managing student information.

## Features

* Add, update, and delete books
* Search books by title or author
* Issue books to students
* Return issued books
* Manage student records

## Technologies Used

* Python
* MySQL

## Project Objective

The main objective of this project is to simplify library management by reducing manual work and maintaining records digitally.

## Future Enhancements

* User login and authentication
* Fine calculation for late returns
* Email notifications
* Report generation
